== Description ==

Turns links to all Qwips into QWiPS Players and lets you attach Qwips to your posts

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `qwips-for-wordpress` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
